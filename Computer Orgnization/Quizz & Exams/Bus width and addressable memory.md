## **Section 1: Multiple Choice Questions (MCQs)**

**1. What was the bus width of the Intel 8088 processor?**  
a) 4 bits  
b) 8 bits  
c) 16 bits  
d) 32 bits  
**Answer: b) 8 bits**

**2. Which processor introduced a 64-bit bus width?**  
a) Intel 80386  
b) Intel 486 DX  
c) Pentium  
d) Intel 8086  
**Answer: c) Pentium**

**3. The Intel 80386DX has an addressable memory limit of:**  
a) 1 MB  
b) 4 GB  
c) 64 GB  
d) 16 MB  
**Answer: b) 4 GB**

**4. Which processor has a 32-bit bus width but only 16 MB of addressable memory?**  
a) Intel 80386SX  
b) Intel Pentium Pro  
c) Intel 80286  
d) Intel Core 2 Duo  
**Answer: a) Intel 80386SX**

**5. What is the addressable memory of the Intel Core i7 EE 990?**  
a) 4 GB  
b) 16 GB  
c) 64 GB  
d) 1 TB  
**Answer: c) 64 GB**

**6. The first microprocessor to contain all components of a CPU on a single chip was:**  
a) Intel 8008  
b) Intel 8080  
c) Intel 4004  
d) Intel 8086  
**Answer: c) Intel 4004**

**7. Which processor had a 16-bit bus width and 1 MB addressable memory?**  
a) Intel 8088  
b) Intel 8086  
c) Intel 80286  
d) Intel 80386  
**Answer: b) Intel 8086**

**8. The Intel 80286 has addressable memory of:**  
a) 1 MB  
b) 4 MB  
c) 16 MB  
d) 4 GB  
**Answer: c) 16 MB**

**9. Which processor has the largest bus width in the 1990s table?**  
a) Pentium  
b) 486 SX  
c) Pentium Pro  
d) Both a and c  
**Answer: d) Both a and c (64-bit)**

**10. The addressable memory of the Intel 4004 is:**  
a) 640 bytes  
b) 16 KB  
c) 64 KB  
d) 1 MB  
**Answer: a) 640 bytes**

## **Section 2: True or False**

1. **The Intel 8080 has a bus width of 8 bits.**  
   **True**

2. **The Intel 80386DX can address up to 16 MB of memory.**  
   **False** (It can address 4 GB)

3. **The Pentium Pro has a 32-bit bus width.**  
   **False** (It has 64-bit bus width)

4. **The Intel 8088 and 8086 both have the same addressable memory (1 MB).**  
   **True**

5. **The Intel Core 2 Duo has a 32-bit bus width.**  
   **False** (It has 64-bit bus width)

6. **The Intel 4004 has a bus width of 4 bits.**  
   **True**

7. **The Intel 80386SX has the same addressable memory as the 80386DX.**  
   **False** (SX has 16 MB, DX has 4 GB)

8. **The Pentium III has a 64-bit bus width and 64 GB addressable memory.**  
   **True**

9. **The Intel 80286 has virtual memory support.**  
   **True** (1 GB virtual memory)

10. **The Intel 486 DX has an addressable memory of 4 GB.**  
    **True**
